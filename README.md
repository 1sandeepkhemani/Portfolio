## Installation & Local Setup

Clone the repository:

```bash
git clone https://github.com/1sandeepkhemani/Portfolio.git
```

Move into the project folder:

```bash
cd Portfolio
```

Run the local preview server:

```bash
npm run start
```

Open in your browser:

```text
http://localhost:5173
```

The local server is required because the website fetches content from:

```text
data/portfolio.json
```
